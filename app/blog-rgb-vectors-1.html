RGB vectors language (11/04/2014)

Been playing for several time with this representation in my free time. This time i wanted to move some interfaces i used at home to web and make some extensions.

Language

Language is a closure over a three float element vector. Each one of these will be mapped to R, G, B values. Common arithmetical operations are supported (performing these for each component of each argument) and some trascendental functions (cos, sin, tan, ...) which are applied for each float value separately. Also sqr, sqrt and createvector for building vector leaf's. Basic variables are x and y, going from 0 to 1. Checkout definition here.

In these examples max-size was always from 30 to 40. 

(VECCOS (VECADD (VECSIN (VECTAN (VECTAN (CREATEVECTOR X 0.5745977 0.26398724)))) (VECDIV (VECSQR (CREATEVECTOR X 0.17096399 X)) (VECMULTIPLY (VECCOS (CREATEVECTOR X Y Y)) (VECCOS (VECABS (CREATEVECTOR Y X X)))))))
