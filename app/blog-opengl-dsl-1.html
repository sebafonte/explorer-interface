OpenGL DSL draft

Last weekend been playing with language defined here . The basic idea was to define a language which basic OpenGL immediate mode functions wrapped, in order to map these functions to a lisp expression. Also alowed some basic control constructs, such as loops.

Not impressive results but it was very important to include iterations as an experience for the test.


Example expression for objects of these kind:

(GL-APP-DRAW (GL-APP-LOAD-IDENTITY) (GL-APP-TRANSLATE-2D 50 50) (GL-APP-POINT-SIZE 2) (GL-APP-REPEAT-N 100 (GL-APP-ROTATE-2D (ABS 50)) (GL-APP-TRANSLATE-2D (GL-APP-REPEAT-VAR *TIME-VARIABLE*) (GL-APP-REPEAT-VAR (+ (ABS 0.95134205) 0.2058494))) (GL-APP-TRIANGLE-2D (GL-APP-REPEAT-VAR 50) (GL-APP-REPEAT-VAR (ABS (ASIN *TIME-VARIABLE*))) 0.4790219 (ABS (ASIN *TIME-VARIABLE*)) (- (GL-APP-REPEAT-VAR (ABS (ASIN *TIME-VARIABLE*))) *TIME-VARIABLE*) (ASIN *TIME-VARIABLE*)) (GL-APP-COLOR 0.93181205 0.96525383 0.021565903)))
