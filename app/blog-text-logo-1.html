Evolving texture coordinates on font textures
Motivation

Needed some logo for a crack-tro like presentation and wanted to make something using just WebGL, some default fonts and some expressions for texture coordinate.

Playing

Started from a initial example (first image) and started to change simple f(x,y,time) functions with my usual mutation interface. Mutation apply a lot of change and it's very hard to evolve, looking why. No fitness evaluation or search algorithm used, just tree random mutations and crossover operations. See example here.
